== Description ==

For many operators who often have to do with the shipments, it will happened several times to avoid the problem
the CAP or incorrect cities submitted by its customers.

The integration of the Wordpress plugin for the Verification code and city allows in a few steps
automation to verify the correct insertion of the cap and the city by a user who is
by making a purchase at a shop e-commerce connected to the WooCommerce plugin.

In a few steps you can set the plugin simply inserting the names from the fields that WooCommerce
use during the checkout fache before confirming the order and during insertion and / or
editing your contact information or shipping addresses.

The plugin will not install or will create database tables that are going to weigh down your installation of wordpress
but it will use the Masses API available freely from RicevitoriaOnLine.it.

You can also insert a small search widget to eventually help users search for a cap or a city,
with the ability to set the fields to display, and help you find the right data.

Within the reserved you will have, also, the possibility of using such research in order to verify if necessary
later that the details are correct to prevent shipments that can remain in storage or denied by couriers.
If you do not have the WooCommerce plugin this can be a tool more to have on his private area to avoid
such problems.


== Installation ==

Upload the wpics plugin to your blog, Activate it, then
configure the plugin for woocommerce and/or the widget if you should use it.
No others configurations or databases are needed
